# ALife Study & Python Practice

## 概要 (Overview)
人工生命（ALife）の学習記録と、基礎力を固めるためのPythonコーディング練習をまとめたリポジトリです。
書籍『作って動かすALife』のコード実装を中心に、様々なシミュレーションの実験を行っています。

## 学習の記録 (Contents)

### 📚 ALife（人工生命）モデルの実装
* **`chap02/` : 空間のダイナミクス**
  * Gray-Scottモデル（反応拡散方程式による模様の生成）
  * 1次元セルラーオートマトン（ルール30などのフラクタル描画）
  * コンウェイのライフゲーム（グライダーガンなど2次元の生存シミュレーション）
* **`chap03/` : 自己組織化と群れ**
  * SCLモデル（自己組織化による細胞形成と膜の発生）※現在実装中

### 🏋️ Python基礎練習
* **`minpy_100knocks/` : みんなのPython 100問チャレンジ**
  * 日々のコーディング基礎固めとして取り組んでいるノート（Jupyter Notebook）

### 🔧 共通ライブラリ
* **`alifebook_lib/`** : シミュレーション描画用の提供ライブラリ
